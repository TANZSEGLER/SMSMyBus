SMSMyBus is a mobile telephone application that lets you find real time bus arrivals for the Madison Metro.

I've turned off the SMS interface due to cost constraints. If you'd like to use SMS, you can always donate to the project using the PayPal link at the bottom, and I'll get you setup.

More ways to find your bus!

Google Talk / Jabber

Jabber your query to smsmybus@appspot.com, and we'll reply back with your results. The syntax is identical to the SMS syntax.

Email

Email your query to request@smsmybus.com with the details in the subject line, and I'll reply with your details.

Are you a developer?

Take advantage of the SMSMyBus API to build your own transit application for the Madison Metro.

The API provides access to real-time arrival estimates, location data, and more. Go check out the documentation.
